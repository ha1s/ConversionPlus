Обработка "Создание правил выгрузки данных".

Обработка предназначена для автоматического создания правил выгрузки данных.

В колонке "Правило конвертации объекта" отображаются существующие правила конвертации объектов. В колонке "Правило выгрузки данных/Имя нового правила выгрузки данных" отображается существующее правила выгрузки данных или, в случае его отсутствия, имя нового правила выгрузки данных. Для создания новых правила выгрузки данных необходимо установить флажки и нажать кнопку "Выполнить".
