Обработка "Загрузка конвертации".

Обработка предназначена для загрузки правил конвертации из XML файла.

В диалоге обработки необходимо указать имя файла загрузки правил обмена и указать наименование новой конвертации. Если идентификаторы конфигураций источника или приемника в загружаемом файле совпадут с существующими в информационной базе, тогда при загрузке правил будет осуществляться поиск объектов, свойств и значений, принадлежащих найденным конфигурациям. Для создания новых конфигураций источника или приемника необходимо для каждой из них включить флаг "Создать новую".
